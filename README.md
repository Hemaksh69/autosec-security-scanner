AutoSec is a lightweight Bash script that performs basic Linux system security checks.
This project was created to learn automation and foundational DevSecOps concepts.

## Features
- Detects if script is running as root
- Lists world-writable files
- Shows top memory-consuming processes
- Displays open listening ports
- Shows kernel & system information
- Generates LOW / MEDIUM / HIGH threat level

## Usage
chmod +x autosec.sh
./autosec.sh

## Purpose
This is a learning-focused project to improve Bash scripting and system auditing basics.
